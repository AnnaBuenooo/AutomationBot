🤖 AutomationBot

Projeto de automação desenvolvido em Python com o objetivo de automatizar tarefas repetitivas no computador, como movimentação do mouse, cliques e digitação automática.

📌 Sobre o projeto

O AutomationBot é um projeto de estudo focado em automação utilizando Python.
Ele utiliza a biblioteca PyAutoGUI para controlar o mouse e o teclado, permitindo automatizar processos que normalmente seriam feitos manualmente.

Esse projeto foi desenvolvido como prática de programação e automação.

🚀 Tecnologias utilizadas

Python

PyAutoGUI

VS Code

Git

💡 Funcionalidades

Automação de tarefas repetitivas

Controle automático do mouse

Automação de cliques

Digitação automática

🎯 Objetivo do projeto

Este projeto foi desenvolvido para:

praticar programação em Python

aprender automação de tarefas

utilizar Git e GitHub no versionamento de código

desenvolver projetos para portfólio
